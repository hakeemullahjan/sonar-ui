# sonar-ui
UI Component libray for Sonar.

## How to Instal
```
npm install @hakeemullahjan/sonar-ui
```

## How to use

```javascript
import { Button } from "@hakeemullahjan/sonar-ui";

function App() {
  return (
    <div className="App">
        <Button label="CLICK ME" />       
    </div>
  );
}

export default App;
```
